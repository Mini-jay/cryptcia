Myzsowski is column transposition algorithm.
We first alphabetise the sentence, and assign ranks to each alphabet based on their position.
If an alphabet appears twice, its given the same rank and they are entered simultaneously.
plaintext is written into a grid horizontally, where number of columns is equal to the length of the keyword.
if only one alphabet, they are read vertically. If many columns have same ranks they are read vertically.
